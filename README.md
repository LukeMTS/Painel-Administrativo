Utilizado:
    - PHP 8.1
    - Composer 2.3
    - MySQL 5.7

1. Fazer uma cópia do arquivo .env.example e renomear para apenas .env
2. Configure o .env como deseja
3. composer install
4. php artisan serve
5. php artisan migrate --seed
6. Acessar localhost:8000
